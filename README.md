# Builder Hexagonal Structure
Allow create folders and classes in PHP with the domain.

# Requirements
```
colorama==0.4.3
```

# Launch

* Copy script builder in your PHP project.

* Create a yml file with the structure
```yaml
    File:
        Size: "int"
        Name : "string"
```

* Run python script 
```
    python builder.py -p ./schema.yml
```
